# Property Listings

A modern South African real estate marketplace built with vanilla web technologies.

## Features

- Hero section with location, type, and price range search
- Responsive property grid with image cards
- Filter and sort functionality (price, bedrooms, property type)
- Property detail modal with full specs
- South African properties across Cape Town, Johannesburg, Durban and more
- Animated stat counters
- Fully responsive mobile-first design

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/ezechias1/property-listings.git
   cd property-listings
   ```

2. Open `index.html` in your browser, or serve with any static server:
   ```bash
   npx serve .
   ```
